The Problems I Encountered and My Solutions

  • Problem3: My website isn't in fully HTML code, it is written in a .md file (Not sure if this is a problem) Solution: I wrote my website in HTML code later on so I didn't use .md files.

  • Problem6: When I tried to open my .html website, it showed a 404 Not Found Page Solution: I opened a new repository with only an index.html file in it. I wrote my HTML code there and made ChatGPT check it so there were no errors in the code. When the home page (index.html) was functioning right, I added the other required pages and made sure there were no errors in their code too.
